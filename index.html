Welcome to phaseFieldFoam’s documentation!¶

phaseFieldFoam is an OpenFOAM implementation of a finite volume solver for coupled phase field equations using the phase field model of Warren, Kobayashi, Lobkovsky and Carter (2003) doi:10.1016/S1359-6454(03)00388-4.

You will find the application source code on Github.

A tutorial case for the growth of a crystal with 6 fold symmetry is found here and documented on Tutorial: Growth of a 2D crystal.
